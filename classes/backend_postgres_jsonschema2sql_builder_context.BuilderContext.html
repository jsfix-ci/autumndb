    • Defined in lib/backend/postgres/jsonschema2sql/builder-context.ts:180

    Get all links stored in this context. The returned map has the following (recursive) structure:

    <links map> = {
  <link type>: [
    [
      {
        nested: <links map>,
        linked: {
            linkName: ...,
            linksAlias: ...,
            joinAlias: ...,
            lateralAlias: ...,
            sqlFilter: ...
        }
      },
      ...
    ],
    ...
  },
  ...
}

    Where nested is optional.

    Returns {}
