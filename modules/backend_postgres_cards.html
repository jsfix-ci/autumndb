Const parseVersionedSlug

  • parseVersionedSlug(slug: string): { base: string; build: string; latest: boolean; major: number; minor: number; patch: number; prerelease: string }
    • Defined in lib/backend/postgres/cards.ts:76

    Parses and normalizes slug versions. Can handle these patterns:

    some-slug - ...@1, ...@1.1, ...@1.2.3 - ...-alpha, ...-alpha+rev1, ...+rev1 - ...@latest and will normalize so that major,minor,patch all fallback to 0 if not specified

    balenaOS versions allow spaces and expect natural sort. This is not valid here. Spaces must be removed and numbers must be padded to be sortable alphanumerically.

    Parameters

    • slug: string

      the slug including the @version part

    Returns { base: string; build: string; latest: boolean; major: number; minor: number; patch: number; prerelease: string }

    the different parts of the versioned slug

    • base: string
    • build: string
    • latest: boolean
    • major: number
    • minor: number
    • patch: number
    • prerelease: string
