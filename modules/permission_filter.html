    summary

    Upsert a card in an unsafe way (DANGEROUS)

    function
    description

    This bypasses the whole permission system, so use with care.

    This function has the added limitation that you can only insert cards of types that are defined in the Jellyfish core.

    example

    const card = await permissionFilter.unsafeUpsertCard(backend, { type: 'foo', links: {}, requires: [], capabilities: [], tags: [], active: true, data: { foo: 'bar' } })

    console.log(card.id)
